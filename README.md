## 🛡️ Troubleshooting

If the dashboard fails to load, check these common items:

*   **Browser window remains blank:** Refresh the page after one minute. The first launch takes extra time to prepare the models.
*   **Window closes instantly:** Ensure you are running the "run_app.bat" file from within the extracted folder.
*   **Interface looks broken:** Resize your browser window. The dashboard adjusts its layout based on your screen width.
*   **Inconsistent data:** Ensure your imported files use the standard CSV format. The dashboard provides a template file in the "data" folder if you need a reference.

## ⚙️ How It Works

The system utilizes Python, a language famous for data analysis. It uses libraries designed for handling large tables of information. The machine learning component examines historical data where customers already left the service. It finds mathematical links between customer traits and the act of leaving. When you input new customer information, the system applies these learned patterns to calculate a score between zero and one. A high score suggests a high risk of churn.

## 📋 Maintaining Quality

The dashboard includes automated checks for your data files. If you load a file with missing information, the software will alert you before it attempts to make a prediction. This prevents errors and ensures your results remain reliable throughout the entire session. You can clear the current data at any time by refreshing the page, which resets the calculation engine to its initial state.
